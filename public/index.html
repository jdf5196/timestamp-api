Timestamp Microservice API

What does this do?

This is a service that converts a readable date passed in the URL to a Unix timestamp and vice versa.

User Stories

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp or a natural language date such as: January 1, 2016.
  2. If id does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp it returns null for those properties.

Url examples:

http://jdf-timestamp-microservice.herokuapp.com/April 8, 2016
http://jdf-timestamp-microservice.herokuapp.com/1460088000

Example Response:

{
  natural: "April 8, 2016",
  unix: "1460088000"
}
